Практичне заняття: Умовні конструкції (C#)

Закріплюємо роботу з if, else if, else та switch.

1 Базовий рівень

1.1 Перевірка віку

Запитай у користувача його вік. Якщо вік ≥ 18 — виведи "Доступ дозволено", інакше — "Доступ заборонено".

Підказка: використовуй int.TryParse, щоб не падати на неправильному вводі.

1.2 Парне чи непарне?

Користувач вводить число. Визнач, чи воно парне.

Використай оператор остачі %: якщо n % 2 == 0 — парне.

2 Середній рівень

2.1 Оцінювання

Користувач вводить кількість балів (0–100). Виведи:

  • 90–100 → "Відмінно"
  • 70–89 → "Добре"
  • 50–69 → "Задовільно"
  • менше 50 → "Незадовільно"
Важливо: умови пиши від більш специфічних до більш загальних.

2.2 Світлофор (switch)

Користувач вводить колір світлофора (red, yellow, green). Виведи відповідну дію:

  • red → "Стоп"
  • yellow → "Приготуйся"
  • green → "Рухайся"

3 Просунутий рівень

3.1 Калькулятор з вибором дії

Запитай два числа та операцію (+, -, *, /). Через switch виконай обчислення. Якщо ділення на нуль — виведи помилку.
